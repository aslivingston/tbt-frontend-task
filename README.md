# Team Directory Widget

An interactive team directory built with React, TypeScript and Vite.

This project demonstrates a dynamic UI component that allows users to browse team members via an autoplaying slider or select a specific person from a dropdown.

---

## Features

- Autoplaying team slider (Swiper)
- Dropdown selector to jump to a specific team member
- Autoplay pauses on selection
- Clear selection to resume autoplay
- Custom navigation controls (top-right arrows)
- Active card styling with gradient/glass effect
- Responsive layout (mobile → desktop)
- Subtle entry animations
- Optional marquee-style team list (experimental)

---

## Tech Stack

- React
- TypeScript
- Vite
- Swiper (slider)
- LESS (styling)
